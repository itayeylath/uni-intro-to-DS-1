# Introduction to Data Science 1

## Course Information

- **Instructor:** Dr. Slava Novgorodov
- **Prerequisites:** 
  - Python programming (CS1 or equivalent)
  - Probability & Statistics (Math1 or equivalent)

## Topics Covered

1. **Introduction** – What is data analysis? What is data science? Ethical considerations
2. **Python Refresher** – Useful commands, Jupyter Notebooks
3. **Data Types** – Structured, unstructured, text, images, audio
4. **Data Manipulation in Python** – Loading, transformations, cleaning, preparation
5. **SQL Basics** – Creating tables, queries, CRUD operations
6. **Text Processing** – Tokenization, text similarity, advanced methods
7. **NLP Tasks** – Sentiment analysis, relation extraction
8. **Data Visualization** – Practical visualization techniques
9. **Web Data Collection** – Web scraping, web crawling
10. **Web Data Analysis** – Extracting insights from internet data
11. **End-to-End Solutions** – Data structuring, merging, cleaning pipelines
12. **Practical Applications** – Real-world use cases
13. **Course Summary & Review**

## Assessment

Final classroom exam
